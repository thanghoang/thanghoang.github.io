\problem{40}
Let $A$ and $B$ are boolean $n\times n$ matrices. The boolean (or and-or) product of $A$ and $B$ is an $n\times n$ matrix $C$ such that

\begin{equation}
	C[i,j]  = \bigvee_k (A[i,k] \wedge B[k,j])
\end{equation}

{\bfseries
\begin{enumerate}
	\item  Reduce boolean matrix multiplication to standard matrix multiplication. 
	Specifically, given a \textsc{SMatMult} subroutine that computes the standard (add-multiply) product of two $n\times n$ matrices in $T(n)$ time, describe and analyze an algorithm \textsc{BMatMult} that multiplies two boolean matrices in $O(T(n))$ time.
	
	\item Let $G=(V,E)$ be an $n$-vertex directed graph. The transitive closure of $G=(V,E)$ is an $n\times n$ matrix $T$ such that for all $(u,v)\in E$
	\begin{equation}
		T[u,v] = \begin{cases}
			1 & \text{if $v$ is reachable from $u$}\\
			0 & \text{otherwise}
		\end{cases}
	\end{equation}
	Describe an algorithm that uses the algorithm \textsc{BoolMatMult} in \textbf{1.A} as the subroutine to compute the transitive closure of $G$ in $O(n^\alpha \log n)$ time, where  $2\le \alpha <3$.
	
\end{enumerate}
}

\problem{40}
You are given a flow network (i.e., a directed graph $G= (V,E)$, a source $s$, a sink $t$, and a positive integral capacity function $c:V\times V\to\naturals$)
along with a flow $f$ in the flow network.
A flow is \textit{acyclic} if the subgraph of directed edges with positive flow contains no directed cycles.

{\bfseries
\begin{enumerate}
    \item  Prove that,
    for any flow $f$,
    there is an acyclic flow with the same value as $f$. (In particular, this implies that some maximum flow is acyclic.)
    
    \item A path flow assigns positive values only to the edges of one simple directed path from $s$ to $t$.
    Prove that every acyclic flow can be written as the sum of $O(E)$ path flows.
    (See flow augmentation on page 678 of CLRS.
    By sum, we mean starting with the zero flow and building up the target flow
    by repeated augmentations.
    Be sure to try some examples of doing this before completing this problem part!)


\item Describe a flow in a directed graph that cannot be written as the sum of path flows.

\item A cycle flow assigns positive values only to the edges of one simple directed cycle. Prove that every flow can be written as the sum of $O(E)$ path flows and cycle flows.


\end{enumerate}
}
