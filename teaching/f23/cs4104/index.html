Resources

  • Main textbook
    • Introduction to Algorithms (4th edition), Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, and Clifford Stein, MIT Press, 2022.
  • Permutation Generation
    • Permutations by interchanges, Heap, The Computer Journal 6.3, 1963
    • Permutation Generation Methods, Robert Sedgewick, ACM Computing Surveys, 1977
  • LaTeX Resources
      • User Manual
    • LaTeX in 24 hours: a practical guide for scientific writing, Dilip Datta, Springer, 2017
    • The Not So Short Introduction to LATEX2ε (v. 5.03), Tobias Oetiker Hubert Partl, Irene Hyna, and Elisabeth Schlegl, 2021
    • How to Insert Graphics into LaTex Document
      • Software
    • Overleaf is a cloud-based LaTeX editor used for writing, editing and publishing scientific documents. It is free through University (required VT email).
    • MiKTeX is a free cross-platform TeX distribution for Windows, Linux and macOS.
    • MacTeX is a free redistribution of TeX Live for specific Mac users.
    • TeXstudio is a free cross-platform IDE to prepare LaTeX document on local machine (Windows, Linux, macOS).
  • LaTeX sample of previous years' homework assignments
