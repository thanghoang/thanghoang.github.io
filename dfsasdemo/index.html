Cattle CARTS: A Coordinated, Auditable, Robust, Traceability System for Beef Industry Data

And how to use it


This supply chain tool implements blockchain metadata storage for its records. Any user can upload a file to be viewed by any number of private access groups. Only members of a given access group, specified by their ownership of that group's private encryption key, will be able to access files uploaded for that group. All actions are logged on the blockchain, such that they are permanent and cannot be edited.
